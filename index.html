A propos de projet

Notre projet consiste à réaliser des visualisations pour le pilotage de l'activité de vente de voitures d'occasions. Pour cela on a utilisé une base de données trouvée sur Kaggle contenant des informations sur différentes voitures d'occasion aux Etats-Unis. Nous avons pu réaliser via cette base de données différentes visualisations qui permettent de combiner,charger,visualiser et explorer cette base de données.

Chaque graphique, tableau ou objet est interactif et s'actualise automatiquement en fonction du contexte associé à chaque action.
